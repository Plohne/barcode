Summary: We collect minimal data. We use Firebase for the app to function and Google Analytics to understand basic usage. No personal data is sold or shared with third parties.

1. Who We Are

This is a personal project - a fridge inventory app hosted at stillbekk.no. The app helps you track products in your fridge using barcode scanning.

2. What Data We Collect

2.1 Data you provide

  • Fridge code - A code you create or enter to access a shared fridge
  • Product information - Names and barcodes of products you add
  • No personal information - We don't collect names, emails, or phone numbers

2.2 Data collected automatically

  • Anonymous user ID - Firebase creates a random ID to enable the app (not linked to you personally)
  • Basic analytics - Page views, app usage patterns (via Google Analytics)
  • Device information - Browser type, screen size (for improving the app)

2.3 Data we do NOT collect

  • ‚ùå Your name or email address
  • ‚ùå Your location (GPS)
  • ‚ùå Your contacts
  • ‚ùå Photos from your device (camera is only used for barcode scanning)

3. How We Use Your Data

  • To make the fridge inventory app work
  • To save your fridge contents so you can access them later
  • To understand how the app is used (so we can improve it)

4. Third-Party Services

4.1 Firebase (Google)

We use Firebase to store app data:

  • Firebase Authentication - Anonymous sign-in (no personal data)
  • Cloud Firestore - Stores fridge and product data

Firebase Privacy Policy: firebase.google.com/support/privacy

4.2 Google Analytics

We use Google Analytics to understand basic app usage. This collects:

  • Pages visited
  • Time spent in app
  • Device/browser type
  • Country (approximate, not precise location)

We have configured Google Analytics to:

  • ‚úÖ Anonymize IP addresses
  • ‚úÖ Disable advertising features
  • ‚úÖ Minimize data retention (14 months)

Google Privacy Policy: policies.google.com/privacy

5. Data Storage & Security

  • Data is stored on Google's servers (Firebase)
  • Data is encrypted in transit (HTTPS)
  • Fridge codes can be password-protected

6. Your Rights (GDPR)

If you're in the EU/EEA, you have the right to:

  • Access - See what data we have about you
  • Delete - Request deletion of your data
  • Opt-out - Decline analytics tracking

To exercise these rights, contact: plohne@github (via GitHub issues)

7. Cookies

We use minimal cookies:

  • Essential cookies - Required for the app to work (Firebase auth)
  • Analytics cookies - Google Analytics (can be declined)

8. Children's Privacy

This app is not directed at children under 13. We do not knowingly collect data from children.
